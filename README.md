# Guardian Eye: AI-Powered Search & Rescue Command System

Guardian Eye is an AI-driven disaster response system designed to assist rescue teams in locating victims, analyzing environmental conditions, and coordinating rescue operations in real time.

---

## 🚀 Features

• Real-time human detection using YOLOv8  
• Injured person detection alerts  
• Thermal visualization of detected persons  
• Depth estimation for safe landing zone detection  
• Environmental hazard analysis  
• Tactical command center dashboard  
• Live webcam / uploaded video analysis  
• VIP target tracking system  
• Real-time alerts with command-center siren system  
• Interactive tactical map with GPS coordinates  

---

## 🧠 Tech Stack

**AI / ML**

## ⚙️ How to Run

### Backend

cd backend/guardian_eye

pip install -r requirements.txt

uvicorn app.main:app --reload


### Frontend

cd rescue-dashboard

npm install

npm start
